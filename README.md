# Youtube Clone developed using React JS and Youtube API V3


### Create Youtube API V3 Key

[https://console.developers.google.com/apis/dashboard](https://console.developers.google.com/apis/dashboard) and select Youtube API V3 App for key.

### Setup Local Instructions

```
git clone https://github.com/praveenorugantitech/praveenorugantitech-youtube-clone-reactjs.git
cd praveenorugantitech-youtube-clone-reactjs

create .env file and include your API Key related to Youtube V3 API

REACT_APP_APIKEY=""


npm install
npm start

```
